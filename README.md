# Projeto-Organizador-de-arquivos-com-.exe

Esse programa acompanha um arquivo .exe que ao ser executado pede para o usuário selecionar uma pasta do computador para ser organizada, ao selecionar uma pasta como "downloads" o programa organiza a pasta movendo cada arquivo para uma pasta com suas especificações, então arquivos .pdf ficaram em uma pasta chamada de "PDF", e fará isso com mais de 25 tipos de arquivos diferentes.
